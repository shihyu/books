歷史
======

本書的版權許可證
------------------

本書以 :doc:`GNU Free Documentation License <app-gfdl>` 發佈，改編和包含了以下兩本書的部分章節。

How To Think Like A Computer Scientist: C++ Version

   作者Allen B. Downey。原書由Green Tea Press發行，可以從 http://www.greenteapress.com/ 下載到。

Programming from the Ground Up: An Introduction to Programming using Linux Assembly Language

   作者Jonathan Bartlett。原書由Bartlett Publishing發行，可以從 http://savannah.nongnu.org/projects/pgubook/ 下載到。

本書的歷史
--------------

2009年2月
   本書首次發佈在網絡上，當時我在亞嵌教育工作，用公司的伺服器，發佈地址是 http://learn.akae.cn/ 。

2009年12月
   在本書的網絡版發行大半年之後，電子工業出版社出版了本書，書名是《Linux C編程一站式學習》（ http://book.douban.com/subject/4141733/ ）。

   本書的網絡版從一開始就聲明以 :doc:`GNU Free Documentation License <app-gfdl>` 發佈，但在國內的出版環境下沒人關心這個，出版社有自己的法律條款。顯然，違反了 :doc:`GNU Free Documentation License <app-gfdl>` 不會有人追究，而違反了出版社的法律條款則是會有人追究的。所以，自從本書與出版社簽了協議之後，網絡版就停止了更新。最終出版的版本在網絡版的基礎上做了很多細節上的修訂，主要是消除歧義和表達得更順暢一些，其中大部分修訂意見來自網絡版的網友留言。

   儘管如此，我還是向出版社爭取到了在網絡上發佈本書最新修訂的權利，但要在本書出版一年之後（也就是2010年12月以後）才可以在網絡上更新。

2010年12月以後
   我始終掛唸著“2010年12月以後允許更新到網絡上的部分”，但我已經換了工作，沒有權限訪問亞嵌教育的伺服器akae.cn了，而且忙於新的工作，所以一直沒有更新網絡版。

2011年3月
   本書加入了一些新的內容，由電子工業出版社重新出版，書名是《一站式學習C編程》（ http://book.douban.com/subject/6025290/ 對豆瓣上9分+的評分我感到十分欣慰，謝謝各位讀者了^^）。新增內容主要有以下幾點：

   *  增加了關於Windows下的C編譯器的介紹，把書名中的“Linux”字眼去掉了，希望更多的讀者能讀到本書。
   *  明確指出書中大部分例子的運行環境是Ubuntu 10.04 LTS 32位x86版，如果讀者也使用同樣的運行環境，一些平台相關的現象應該會和書中的描述一致。
   *  “Makefile基礎”一章中“自動處理標頭檔依賴關係”一節，我結合工作經驗給出一個更好的解法，和上一版（《Linux C編程一站式學習》）的該小節完全不同。

   很遺憾，出版社不同意在網絡上更新這些新增內容，即使在出版一年之後也不行。

2012年7月
   我辭去了工作準備出國，終於有時間處理“2010年12月以後允許更新到網絡上的部分”了。根據之前和出版社的約定，《一站式學習C編程》的新增內容不能更新到網絡上，所以我決定放棄這些內容，繼續在原來的《Linux C編程一站式學習》的基礎上更新。

   我在自己的伺服器 http://songjinshan.com/ 上重新發佈本書，最新版本不涉及Windows下的C編譯器的內容，將運行環境設定為最新的Ubuntu 12.04 LTS 32位x86版，至于“Makefile基礎”一章中“自動處理標頭檔依賴關係”一節，我給出了完全不同於前兩種解法的第三種解法。總之，最新版本不包含《一站式學習C編程》裡的任何修訂內容和新增內容。此外，最新版本包含英文版，我的英文寫作水平一般般，為國外的生活做準備練練筆而已，獻醜了。英文版和中文版並非一一對譯的，但英文版肯定會覆蓋中文版的所有技術點。
